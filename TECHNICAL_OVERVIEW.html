Technical Architecture & Development Portfolio

Avanza Pathfinders is a production-grade career platform that uses AI and modern web architecture to bridge the gap between global talent and the Italian economy.

🏗️ Technological Stack

Layer Technology Strategic Benefit
Frontend React.js + Vite Extreme speed and responsive UI components.
Backend FastAPI (Python) High-performance logic for AI and data parsing.
Authentication Stytch SDK Secure, passwordless "Magic Link" experience.
Database Supabase (PostgreSQL) Enterprise-grade data security and scalability.
Deployment Docker + Vercel + Render Professional containerization and CI/CD automation.

🗺️ Core Architecture

The platform follows a Decoupled Monorepo model, ensuring that the frontend interface and the AI brain can scale independently while sharing the same security protocols.

1. The Global Localization Engine

A custom-built localization system supporting 7 languages simultaneously. This architecture allows the platform to be "Global First," ready to serve non-Italian speakers from Day 1.

2. Cloud Infrastructure

Hosted across a distributed network for 99.9% uptime:

  • Frontend: Edge-delivered via Vercel for instant loading worldwide.
  • Backend: Containerized via Docker on Render for robust reliability.
  • Domain: Custom-mapped to avanza.it.com using advanced DNS records.

🚀 Future Scope

The current code supports a immediate rollout of:

  • AI CV Parsing: Automating the extraction of professional skills.
  • Recognition Tracker: Real-time mapping of university degree legalization steps.
  • Market IQ: Live integration with the Adzuna API for Italian job trends.
